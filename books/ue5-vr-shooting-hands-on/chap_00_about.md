# この本について
この本はUE5を使った初心者向けVRゲーム開発ハンズオンで使用するために書いたテキストです。
ちょっとしたVRゲームの作成を通じて、UE5を使ったVR開発の基礎的な部分に触れることを目的としています。

# 対象バージョン
- UE5.3.2
- Windows 10 / 11

# 作るゲーム
射撃訓練場（キルハウス）のゲームです。

https://youtu.be/zKUHxLRzbjE

射撃訓練場は実際に発売されているFPSやVRのゲームでも練習モードでよく見かけます。

https://youtu.be/WUf8tXL-PnU
